Guide d'utilisation :
ATTENTION -> Avant toute chose, NE JAMAIS supprimer manuellement les fichiers non cryptés utilisez uniquement le bouton "remove decrypted files" pour cela... 
1:
en premier lieu, placez vos fichiers à crypter dans le dossier "decrypted". Ensuite, cliquez sur le bouton "encrypt" et entrez un mot de passe (pour obtenir un mot de passe sécurisé, consultez https://www.economie.gouv.fr/particuliers/creer-mot-passe-securise). Ensuite, vérifiez qu'un fichier avec l'extension .ElapsEncryption a été généré dans le dossier "encrypted". Vous pouvez alors cliquer sur le bouton "remove decrypted files" pour supprimer les fichiers non cryptés de manière sécurisée.
2:
Lorsque vous souhaitez récupérer vos fichiers cryptés, cliquez sur "decrypt" puis entrez votre mot de passe. Les fichiers seront alors copiés vers le dossier "decrypted" et vous pouvez les consulter. Ensuite, lorsque vous souhaitez arrêter de les consulter, cliquez sur le bouton "remove decrypted files". (Vous n'avez pas besoin de cliquer sur "encrypt" avant de supprimer car les fichiers cryptés sont déjà stockés dans le dossier "crypted".)

Remarque sur la Sécurité :

   - Si vous oubliez votre mot de passe, vous ne pourrez jamais récupérer vos fichiers chiffrés.

   - La suppression sécurisée de fichiers utilise des méthodes de réécriture pour garantir que les données sont irrécupérables. Soyez sûr de vouloir supprimer les fichiers avant de procéder.
