User Guide:

WARNING -> First of all, NEVER manually delete unencrypted files; use the "Remove Decrypted Files" button for this...

1:
Firstly, place your files to be encrypted in the "decrypted" folder. Then, click on the "Encrypt" button and enter a password (to get a secure password, check https://www.youtube.com/watch?v=0RLH4SNAfjA). Then, verify that a file with the .ElapsEncryption extension has been generated in the "encrypted" folder. You can then click on the "Remove Decrypted Files" button to securely delete the unencrypted files.

2:
When you want to retrieve your encrypted files, click on "Decrypt" and enter your password. The files will then be copied to the "decrypted" folder, and you can view them. When you want to stop viewing them, click the "Remove Decrypted Files" button. (You do not need to click "Encrypt" before deleting because the encrypted files are already stored in the "encrypted" folder).

Security Note:

If you forget your password, you will not be able to recover your encrypted files.

Secure file deletion uses overwrite methods to ensure that the data is unrecoverable. Make sure you want to delete the files before proceeding.
